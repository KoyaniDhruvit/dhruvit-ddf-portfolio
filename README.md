## 2. Task 1: LED Control Circuit

### 2.1 Simple LED Circuit (Task 1.1)

| R1 [Ω] | Measured V1 [V] | Measured V_LED [V] |
| :--- | :--- | :--- |
| 220 | 2.28 | 2.86 |
| 1000 | 2.77 | 2.51 |
| 4700 | 2.97 | 2.31 |

**Observations:**  
As resistance increases, LED brightness decreases. The voltage distribution changes across the circuit, affecting LED operation.

### 2.2 Switchable LED Circuit (Task 1.2)

The switch controls whether the circuit is complete or open.

- Closed → LED ON  
- Open → LED OFF  

A switch works in both directions because it has no polarity.

### 2.3 Dimmable LED Circuit (Task 1.3)

| Position | V_LED [V] | V2 [V] |
| :--- | :--- | :--- |
| Full Brightness | 3.01 | 3.05 |
| Dimmed | 2.30 | 4.45 |
| OFF | 0.005 | 4.50 |

The potentiometer acts as a voltage divider, allowing smooth brightness control.
